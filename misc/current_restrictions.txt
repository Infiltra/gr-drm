### Gnu Radio DRM Transmitter ###

CURRENT RESTRICTIONS: (last updated July 2 2012)

- DRM+ only prepared, not functional
	- correct FAC init missing
	- AFS reference cells missing
- Hierarchical mapping only prepared, not functional
- no multiple services supported, only 1 audio stream (AAC mono, no SBR, no MPEG surround) allowed
	- therefore no FAC repetition patterns implemented
- reconfiguration not supported
- no audio CA supported
- language is set to German, programme type to 'Science'
- no multiple SDC blocks supported
- a fixed set of SDC data entities is transmitted (type 0,1,8,9)
